## How to install / launch it

```sh
git clone <to-define>

# If you are on Nix
# nix-shell

cp .env.default .env

docker-compose up -d
```

Please don't forget to setup the env file before launching the docker-compose.
The default .env file is this one [.env.default](./.env.default).
